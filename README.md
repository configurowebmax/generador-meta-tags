# Generador de Meta Tags SEO Gratis

Aplicación web gratuita para **generar meta tags SEO y Open Graph** para tu sitio web.

## ✨ Características

- Genera tags: **title, description, keywords, robots, author**.
- **Open Graph** (Facebook, LinkedIn, WhatsApp).
- **Twitter Cards** opcional.
- Contadores de caracteres (60/160 recomendado).
- Copiar código con un clic.

## 🚀 Uso

1. Completa los campos (título, descripción, URL, imagen).
2. Copia el código generado en el `<head>` de tu web.
